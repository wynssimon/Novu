# Voor je de applicatie kan gebruiken!
## Maak een account op Novu.com
## Maak twee workflows:
- SMS workflow
- Email workflow

Kopieer beide id's en plak deze samen met je API key in een .env file
zoals:
    NOVU_API_KEY=...
    NOVU_EMAIL_WORKFLOW_ID=...
    NOVU_SMS_WORKFLOW_ID=...


Edit je workflows zodat er een mail of sms gestuurd wordt bij de trigger.
